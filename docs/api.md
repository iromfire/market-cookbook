## HTTP (Fetch/XHR в DevTools)

✅ Работай через сгенерированные репозитории, модели и enum'ы из `@ui-api`

```
@ui-api/
├── repositories/
├── models/
└── enums/
```

❌ Не описывай API вручную. Создавай свои модели и enum'ы в крайнем случае

✅ Используй нейминг BE

❌ Не придумывай свой нейминг

### Обновление

```bash
npm run swagger:sync:<stable | servicing | trunk>
```

✅ Выноси генерацию API в отдельный PR

❌ Не смешивай генерацию API и разработку в одном PR

::: tip ℹ️

Тип `Blob` указывается на BE по умолчанию как возвращаемый тип методов.  
Если нужна корректная типизация — обратись к BE, чтобы поправили контракт в Swagger

:::
